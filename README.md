BBS (Bulletin Board System) — это электронная доска объявлений. По сути, это компьютер, работавший как сервер, к которому другие пользователи могли подключаться через телефонную линию (с помощью модема) для обмена сообщениями, файлами, играми и общения. Это предшественник современных форумов, чатов и соцсетей. Золотая эра BBS пришлась на 1980-е — середину 1990-х годов.
